Welcome to the mobile app of "Interactive Phonetics".

In learning phonetics, it is necessary to learn a set of symbols with which one may notate speech sounds. The most commonly used alphabet for this purpose is the International Phonetic Alphabet (IPA) developed by the International Phonetic Association (established 1886 in Paris).

The IPA contains a list of symbols (phonetic letters). Each of these letters represents an unit of linguistic sound known as a phone/segment.
This list is presented in a table known as the IPA chart.
